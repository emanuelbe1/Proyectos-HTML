Meta viewport.

El Viewport es una de las etiquetas más representativas de la web móvil, que nos permite configurar el cómo debe interpretar una página el navegador web para móviles.

Media queries.

Las media queries(consulta de medios) son útiles cuando deseamos modificar la página web o aplicación en función del tipo de dispositivo (como impresora o pantalla) o de características y parámetros específicos (como resolución de la pantalla o el ancho del viewport del navegador)


CSS y Frameworks, resumen.

  • Meta viewport: define el ancho de pantalla por tomar.
  • Media queries: sirven para abarcar diferentes tamaños de pantalla.
  • Propiedad Max-width: sirve para no deformar las imágenes.
  • Box model: pensar todo como en cajas.
  • Bootstrap: uno de los frameworks más populares.

Introducción a CSS.

En este módulo:

  • Aplicar estilos a documentos HTML.
  • Capa de presentación, aplicada a través de CSS, para customizar documentos.
  • Distintas formas de aplicar CSS: buenas prácticas, box model y conceptos clave.

Definición de CSS:

CSS(Hoja de Estilo en Cascada) es el código para dar estilo a la página web. Este no es un lenguaje de programación o de marcado; es un lenguaje de hojas de estilo, y permite aplicar estos de forma selectiva a los elementos en documentos HTML.

Tipos de aplicación de estilos en CSS

  • Estilos Inline.
  • Estilos internos.
  • Incluir un archivo de CSS externo.(Considerada buena practica)

Unidades de medida en CSS

  • px: pixeles
  • em: unidad de medida, 1 em == tamaño de fuente actual.
  • rem: unidad de medida 1 rem == tamaño de fuente del cuerpo(body)
  • porcentaje, representado por el signo %.

Colores en CSS

Hay tres tipos de notaciones para el color:

  • Hexa: Valores hexadecimales, "#ff0000".
  • RGB: Valor en la escala RGB, "rgb (255, 0, 0)".
  • Nombre: El nombre del color en inglés como "red".

Selectores por elementos

Ejemplo de aplicación de estilo a los elementos <h1>, <h2>, y <p>

 h1, h2, p {text-align: center;
           color: red;}

Selectores por ID

Ejemplo de aplicación de estilo por ID

 #parraf1 {text-align: center;
           color: red;}

·En HTML:

<p id="parraf1">Texto de ejemplo</p>
